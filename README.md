# 🛠️ 办公自动化工具箱 v3.0

在线版办公自动化工具，无需安装，浏览器直接使用。

## 功能

- 📊 合并多个 Excel 文件
- 🧾 提取 PDF 发票信息
- 📧 发送邮件（带附件）
- 📈 批量处理 Excel（去重/筛选/统计）
- 🗂️ 文件整理（按类型归档下载）

## 本地运行

```bash
pip install -r requirements.txt
python app_cloud.py
```

打开 http://127.0.0.1:5000

## 云部署

支持 Render / Railway / PythonAnywhere 等平台一键部署。
